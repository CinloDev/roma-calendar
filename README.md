# Proyecto Turnos (fork de Roma Turnos)

Repositorio inicial para clonar y mejorar https://roma-turnos.netlify.app/.

Estructura:
- `frontend/` - código frontend (Next.js/React recomendado)
- `backend/` - funciones/servidor y scripts (Supabase/Edge Functions o Node)

Pasos rápidos:
1. Crear proyecto en Supabase (Postgres + Auth) o usar Render para backend.
2. Configurar variables de entorno en `frontend/.env.local` y `backend/.env`.
3. Desplegar frontend en Vercel/Netlify y backend en Supabase/Render.

Ver `frontend/README.md` y `backend/README.md` para instrucciones específicas.
